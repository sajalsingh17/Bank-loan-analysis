 Bank-loan-analysis
This project focuses on analyzing bank loan data using SQL, Excel, and Power BI to gain insights into loan performance, customer behavior, and risk assessment. The objective is to transform raw financial data into meaningful dashboards and reports that help in making data-driven decisions for banking and financial institutions.
Here’s a clean, professional GitHub project description you can directly use or slightly customize for your repository on Bank Loan Analysis using Power BI, Excel, and SQL:

Tools & Technologies Used:

SQL – Data extraction, cleaning, and transformation

Excel – Data preprocessing, pivot tables, and basic analysis

Power BI – Interactive dashboards and data visualization,DAX Formula

Key Objectives:

Analyze loan distribution by customer demographics

Identify trends in loan approval, rejection, and default rates

Evaluate interest rates and loan settlement performance

Segment customers based on income, age, employment status, and credit risk

Build an interactive dashboard for business decision-making

Features:

Cleaned and structured dataset

SQL queries for filtering and aggregating data

Excel sheets for validation and analysis

Power BI dashboards with KPIs and charts:

Total Loan Amount

Default Rate

Approval Rate

Loan Status Breakdown

Customer Risk Analysis
